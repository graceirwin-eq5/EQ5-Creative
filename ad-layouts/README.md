# Ad layout concepts — 1080×1080

Four eye-catching square ad layout designs, each containing the same asset set arranged across different color moods and geometric pattern systems.

## Files

| File | Concept | Color mood | Pattern motif |
| --- | --- | --- | --- |
| `01-bold-vibrant-diagonal.svg` | Bold & vibrant | Cream / coral / teal | Diagonal slash + hatching |
| `02-midnight-concentric.svg` | Dark & sophisticated | Black / amber | Concentric rings + halftone dots |
| `03-earthy-arches.svg` | Warm & earthy | Terracotta / mustard / cream | Nested semicircle arches |
| `04-modernist-bauhaus.svg` | Modernist | Mint / black / red / amber | Grid + primary shapes |

## Asset placements (consistent across all four)

- **Logo** — top-left
- **Headline** — two-line, mid-card
- **Body copy** — three weighted lines below headline
- **CTA** — pill or square button
- **Product hero** — varied position to balance the geometric pattern

## Notes

- Native dimensions: 1080×1080 (Instagram / Facebook feed square).
- All SVGs use absolute coordinates and embedded patterns — no external assets, fonts fall back to system sans/serif.
- For production use: swap the placeholder logo wordmarks, product silhouettes, and copy bars with real brand assets and copy.
