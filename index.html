What's working

Strong hook in the first 3 seconds. Minimal copy keeps focus on the brand claim. Color contrast drives thumb-stop in feed.

Strategy recommendations

Test a version with a stat headline. Try a darker colorway for the prescribing segment. A/B the CTA copy.

01
Creative Brief Development
  • Create brief document for internal reference to be stored in Google Drive and sent to those on account
  • Creative kickoff call complete
  • Make updates and approve brief document
  • Align on brand guidelines and assets
  • Align on company rules for MLR
02
Draft Creative Mockup of Ads
  • Matrix input
  • Provide Claude with content to be referenced/leveraged from approved banners and website
  • Claude to compare Fireflies, Slack, and email content to matrix
03
Art to Draft Layouts
  • Claude to offer design ideation/variation
  • Export layouts from Claude to Figma
  • Revise layouts in Figma / rework assets
  • Claude to check layouts against matrix
04
Internal Reviews
  • Upload unmarked Figma file to Claude
  • Set internal review meeting with core team
  • Art to make comments in Figma for proposed changes
05
Art to Implement Revisions
  • Upload marked Figma file to Claude for Claude to read
  • Claude to itemize proposed changes
  • Make edits
  • Upload revised Figma file to Claude as most recent copy to reference
06
AM to Send to Client Review
  • AM to set up client review meeting (if applicable)
07
Art to Implement Client Feedback
  • Claude to itemize new revisions to be made
  • Make revisions
  • Upload revised Figma file and have Claude compare to client feedback
08
Client Approval for MLR Submission
  • For multi-page, heavy content PDF development:
  • Create in mockups design system in Claude
  • Instruct layout edits and rework where needed
  • If too difficult, switch to Figma
  • Use Claude to place ads in mockups using design system
  • Input all necessary information/assets as labelled in template and command to place in mockup PDF
  • Clarify what info is what for template to read and place accurately
  • Clarify which social channels to be used in mockup
  • If multipage PDF, clarify ad layouts to be separated by page
  • Once PDF looks right, ask Claude to open PDF in new tab in order to save
  • Once open in new tab, Cmd+P to save
  • For multipage PDF files, export mockup PDF as SVG into Figma
  • Make edits and export in Figma
  • Return to Claude to check placements
09
MLR Submission
  • Submit creatives for MLR review
  • Track MLR review date
  • AM to review and markup MLR comments
  • Art to implement comments
  • Claude to compare new layout to comments
  • Internal QA / Approval
  • Client Review
  • Submit for final approval / AFD
  • Approval for distribution received
